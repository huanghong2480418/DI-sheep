## 一🌸常见问题🌸一

1.无法访问此网站链接已重置：

①多次刷新（至少1分钟）

②使用梯子

2.出现Github404页面：

①可能是号没了，或者是关闭了

②网址打错了，注意区分大小写

3.没有声音：

①断开与蓝牙耳机的连接，重新访问

②刷新网站有几率恢复

③使用梯子

4.卡在加载页面：

①耐心等待

②多次刷新

③使用梯子
